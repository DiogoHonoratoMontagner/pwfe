FUNDAMENTOS DE PROGRAMAÇÃO À OBJETOS

O que é Programação Orientada a Objetos?

Programação orientada a objetos é um paradigma de programação baseado no conceito de "objetos", que podem conter dados na forma de campos, também conhecidos como atributos, e códigos, na forma de procedimentos, também conhecidos como métodos. Uma característica de objetos é que um procedimento de objeto pode acessar, e geralmente modificar, os campos de dados do objeto com o qual eles estão associados.

Por que usar Orientação a Objetos?

É mais fácil pensar em objetos do que em funções / procedimentos pois objetos estão mais próximos ao mundo real. A orientação a objetos trás vários recursos para produzir componentes independentes intercambiáveis (encapsulamento, interfaces, etc), promovendo o baixo acoplamento.

O mercado para a Programação Orientada a Objetos

Resultado de imagem para o mercado para a programação orientada a objetos. Hoje em dia, o mercado de programação é dominado pelo paradigma orientado a objetos. Em outras palavras, é raro você trabalhar com uma linguagem de programação atual que não suporte POO. Saber POO não é um capricho, mas sim uma exigência do mercado.
